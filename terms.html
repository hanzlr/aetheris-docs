Syarat & Ketentuan Layanan

Selamat datang di UMB E-SPORT BOT ("Bot", "kami"). Dengan menambahkan, mengakses, atau menggunakan Bot ini di server Discord manapun, kamu ("pengguna", "kamu") dianggap telah membaca, memahami, dan menyetujui seluruh Syarat & Ketentuan berikut. Jika kamu tidak setuju dengan salah satu ketentuan di bawah, harap tidak menggunakan Bot ini.

1. Tentang Layanan

UMB E-SPORT BOT adalah bot Discord yang dikembangkan untuk komunitas UMB E-SPORT, menyediakan berbagai fitur hiburan dan utilitas server, termasuk namun tidak terbatas pada:

  • Sistem tiket bantuan (Open Ticket)
  • Leveling, XP, dan Ranking
  • Mata uang virtual, Daily Reward, dan sistem Bank dengan bunga mingguan
  • Mini game (Dice, Coinflip, Slot, Blackjack)
  • Profile Card, Badge, dan Statistik pengguna
  • Loot Box, Fishing, Inventory, dan Shop System
  • Sistem Equip, Boost, dan Event
  • Web Dashboard beserta Admin Tools
  • Fitur tambahan seperti Roast Me dan Help Command

Bot ini terus dalam pengembangan, dan fitur baru dapat ditambahkan, diubah, atau dihentikan dari waktu ke waktu tanpa pemberitahuan sebelumnya.

2. Persyaratan Usia & Kepatuhan pada Discord

Penggunaan Bot ini tunduk pada Discord Terms of Service dan Discord Community Guidelines. Kamu harus memenuhi syarat usia minimum yang ditetapkan Discord untuk memiliki akun dan menggunakan layanan Discord.

3. Mata Uang Virtual & Mini Game

Seluruh mata uang, item, badge, loot box, dan aset virtual lainnya dalam Bot ini hanya berlaku di dalam ekosistem Bot dan tidak memiliki nilai uang sungguhan. Aset ini tidak dapat ditukar, dijual, atau dikonversi menjadi uang tunai atau barang/jasa di dunia nyata.

Mini game seperti Dice, Coinflip, Slot, dan Blackjack disediakan murni sebagai hiburan dalam bentuk simulasi menggunakan mata uang virtual. Mini game ini bukan perjudian karena tidak melibatkan transaksi uang sungguhan dalam bentuk apapun, baik untuk bertaruh maupun untuk menerima hasil.

4. Tata Tertib Pengguna

Saat menggunakan Bot, kamu setuju untuk tidak:

  • Mengeksploitasi bug, celah sistem, atau kerentanan apapun pada Bot untuk keuntungan tidak adil (misalnya menggandakan mata uang atau item);
  • Menggunakan automation, self-bot, script, atau alat pihak ketiga apapun untuk berinteraksi dengan Bot;
  • Menyalahgunakan sistem tiket, leveling, ekonomi, atau fitur lainnya dengan tujuan spam atau merugikan pengguna lain;
  • Mencoba mengakses, merusak, atau mengganggu jalannya Bot maupun Web Dashboard tanpa izin.

Pelanggaran terhadap ketentuan ini dapat berakibat pada pencabutan akses ke fitur Bot, penghapusan data progres (mata uang, level, inventory), atau pemblokiran dari penggunaan Bot, atas kebijakan tim pengembang.

5. Hak Kekayaan Intelektual

Seluruh kode, desain, branding, dan konten orisinal dari UMB E-SPORT BOT adalah milik tim pengembang, hanz (Nexalab). Kamu tidak diperkenankan menyalin, mendistribusikan ulang, atau membuat turunan dari Bot ini tanpa izin tertulis dari tim pengembang.

6. Web Dashboard & Admin Tools

Akses ke Web Dashboard dan Admin Tools hanya diberikan kepada pihak yang berwenang (admin/moderator server terkait). Penyalahgunaan akses admin, termasuk mengubah data pengguna lain tanpa alasan yang sah, dapat dikenakan sanksi sesuai kebijakan tim pengembang.

7. Batasan Tanggung Jawab

Bot ini disediakan "sebagaimana adanya" (as-is) tanpa jaminan apapun, baik tersurat maupun tersirat, termasuk namun tidak terbatas pada jaminan ketersediaan, keakuratan data, atau bebas dari gangguan teknis. Kami tidak bertanggung jawab atas kerugian, kehilangan data, atau dampak apapun yang timbul dari penggunaan atau ketidakmampuan menggunakan Bot ini.

8. Perubahan Layanan & Ketentuan

Kami berhak mengubah, menambah, atau menghentikan fitur Bot kapan saja. Syarat & Ketentuan ini juga dapat diperbarui dari waktu ke waktu; versi terbaru akan selalu tersedia di halaman ini. Dengan tetap menggunakan Bot setelah perubahan diberlakukan, kamu dianggap menyetujui ketentuan yang telah diperbarui.
